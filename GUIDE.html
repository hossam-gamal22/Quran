📦 التسطيب — خطوة بخطوة
ℹ️
المتطلبات الأساسية: Node.js 18+ ، Git ، VS Code ، وجهاز Android أو iPhone أو محاكي

الخطوة 1: تثبيت Node.js

  1. تحميل Node.js LTS
    اذهب إلى nodejs.org واختر نسخة LTS (18 أو 20)
    # تحقق من التثبيت node --version # يجب أن تظهر v18+ npm --version # يجب أن تظهر 9+
  2. تثبيت pnpm (مُوصى به)
    أسرع وأكثر كفاءة من npm
    npm install -g pnpm pnpm --version # تحقق من التثبيت

الخطوة 2: استخراج المشروع

  1. فك ضغط الملف
    فك ضغط quran-app-v7-final.zip في مكان مناسب
    # مثال على Windows Expand-Archive quran-app-v7-final.zip -DestinationPath C:\Projects\ # مثال على Mac/Linux unzip quran-app-v7-final.zip -d ~/Projects/
  2. الانتقال لمجلد المشروع
    المشروع داخل مجلد "improved"
    cd Projects/improved
  3. تثبيت المتطلبات
    هذا سيستغرق 2-5 دقائق
    pnpm install # أو بدون pnpm: npm install
💻 إعداد VS Code
⬇️
تحميل VS Code: code.visualstudio.com — مجاني ومتوفر لكل الأنظمة

  3. افتح المشروع في VS Code
    من داخل terminal أو بالسحب والإفلات
    # من Terminal code . # أو File → Open Folder → اختر مجلد improved/
  4. إعدادات VS Code للمشروع
    أنشئ ملف .vscode/settings.json

    لتتمكن VS Code من رؤية الأنواع بشكل صحيح
    npm install -g typescript ts-node
▶️ تشغيل التطبيق

🌐 في المتصفح (أسرع للتطوير)

  1. شغّل التطبيق
    pnpm dev # أو: npx expo start --web
  2. افتح في المتصفح
    اضغط W في terminal أو افتح localhost:8081

📱 على جهازك الحقيقي

  1. ثبّت تطبيق Expo Go على هاتفك
    متاح على Google Play وApp Store مجاناً
  2. تأكد أن هاتفك وحاسوبك على نفس الـ WiFi
    npx expo start # امسح QR Code من Expo Go على هاتفك

🤖 على Android Emulator

  1. ثبّت Android Studio
    developer.android.com/studio — مجاني
  2. أنشئ Virtual Device (AVD)
    Android Studio → Device Manager → Create Device → Pixel 7 / API 34
  3. شغّل المحاكي ثم التطبيق
    npx expo run:android # أو اضغط A في terminal بعد expo start

🍎 على iOS Simulator (Mac فقط)

  1. ثبّت Xcode من App Store
    npx expo run:ios # أو اضغط I في terminal
⚡
أوامر VS Code المفيدة:
• Ctrl + `` ` `` — فتح Terminal مدمج
• Ctrl + Shift + P — Command Palette
• F5 — تشغيل مع debugger (بعد إعداد launch.json)
• Ctrl + Click على اسم ملف — فتح الملف
🔨 بناء التطبيق للإنتاج
  1. تثبيت EAS CLI
    npm install -g eas-cli eas login # سجّل دخولك بحساب expo.dev
  2. تهيئة المشروع لـ EAS
    eas build:configure # سيُنشئ ملف eas.json
  3. تحديث app.json — معلومات مهمة
    { "expo": { "name": "القرآن الكريم", "slug": "quran-app-hossam", "version": "1.0.0", "android": { "package": "com.hossamgamal.quranapp", ← غيّر هذا "versionCode": 1 }, "ios": { "bundleIdentifier": "com.hossamgamal.quranapp" ← غيّر هذا } } }
  4. بناء Android (.aab)
    eas build --platform android --profile production # يستغرق 10-20 دقيقة على السحابة
  5. بناء iOS (.ipa)
    eas build --platform ios --profile production # يحتاج Apple Developer Account (99$/سنة)

تغيير ألوان الثيم

لتغيير اللون الرئيسي للتطبيق، عدّل ملف theme.config.js:

// theme.config.js const themeColors = { primary: { light: '#1B6B3A', ← غيّر هذا للوضع الفاتح dark: '#2ECC71', ← غيّر هذا للوضع الداكن }, // باقي الألوان... };

بعد التغيير، أعد تشغيل npx expo start لرؤية التغييرات. الألوان المقترحة:
