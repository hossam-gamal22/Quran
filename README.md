# 🕌 القرآن الكريم - Quran App

تطبيق قرآن كريم متكامل مبني بـ **React Native + Expo**

---

## 📦 المتطلبات

- **Node.js** v18+
- **pnpm** v9.12.0 — `npm install -g pnpm@9.12.0`
- **Expo CLI** — `npm install -g expo-cli`
- **VS Code** مع امتداد **Expo Tools**

---

## 🚀 تشغيل المشروع

```bash
# 1. تثبيت المكتبات
pnpm install

# 2. تشغيل التطبيق
pnpm dev

# 3. تشغيل على Android
pnpm android

# 4. تشغيل على iOS (Mac only)
pnpm ios
```

## 📱 الميزات

| الميزة | الوصف |
|--------|-------|
| 📖 **قراءة القرآن** | 114 سورة بالنص العثماني |
| 🔊 **التلاوة الصوتية** | 8 قراء مختلفين، تشغيل متواصل |
| 🌍 **الترجمات** | 10 لغات + تفاسير عربية |
| 📚 **التفسير** | الميسر، الجلالين، Sahih International |
| 🕌 **أوقات الصلاة** | بناءً على الموقع الجغرافي |
| 🧭 **اتجاه القبلة** | بوصلة حية |
| 🔖 **المفضلة** | حفظ الآيات |
| 🌙 **الوضع الداكن** | دعم كامل للـ Dark Mode |

## 🗂️ هيكل المشروع

```
quran-app/
├── app/
│   ├── (tabs)/
│   │   ├── index.tsx        # الصفحة الرئيسية
│   │   ├── quran.tsx        # القرآن الكريم
│   │   ├── prayer.tsx       # أوقات الصلاة
│   │   ├── qibla.tsx        # اتجاه القبلة
│   │   └── settings.tsx     # الإعدادات
│   ├── surah/
│   │   └── surah-screen.tsx # شاشة قراءة السورة
│   └── tafsir/
│       └── [surah]/[ayah].tsx # شاشة التفسير
├── lib/
│   ├── quran-api.ts         # واجهة AlQuran.cloud API
│   ├── prayer-api.ts        # واجهة AlAdhan API
│   ├── storage.ts           # التخزين المحلي
│   ├── settings-context.tsx # إعدادات التطبيق
│   └── theme-provider.tsx   # مزود الثيم
├── constants/
│   └── theme.ts             # ألوان التطبيق
└── assets/
    └── images/              # الأيقونات والصور
```

## 🔧 APIs المستخدمة

- **AlQuran.cloud** — النص القرآني والترجمات والبحث
- **AlAdhan** — أوقات الصلاة واتجاه القبلة
- **Islamic.Network CDN** — ملفات صوتية للتلاوات
