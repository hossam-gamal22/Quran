# Sound Files — روح المسلم

## Folder Structure

```
assets/sounds/
├── adhan/            # Adhan (call to prayer) sound files
├── notifications/    # Notification sound files
├── adhkar/           # Adhkar narration audio files
├── effects/          # UI sound effects
└── README.md
```

## Naming Conventions

- Use **lowercase kebab-case** for all filenames: `salawat.mp3`, `tap-complete.mp3`
- Preferred format: **MP3** (`.mp3`) for broad compatibility
- Keep file sizes small — compress audio for mobile delivery

## Folder Details

### `adhan/`
Adhan recordings for prayer time notifications.
- Example: `makkah.mp3`, `madinah.mp3`, `alaqsa.mp3`, `mishary.mp3`, `abdul-basit.mp3`

### `notifications/`
Short sound clips used for scheduled notification alerts.
- `salawat.mp3` — الصلاة على النبي
- `istighfar.mp3` — الاستغفار
- `subhanallah.mp3` — التسبيح

### `adhkar/`
Full narration audio for adhkar content.

### `effects/`
UI interaction sound effects (tap feedback, completion chimes, etc.).
- Example: `tap.mp3`, `complete.mp3`
